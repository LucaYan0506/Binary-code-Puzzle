## How to play the game
* Click "The rules of the puzzle" to see the rule  
<div style="float">
<img src="https://github.com/LucaYan0506/Binary-code-Puzzle/blob/master/screenshot/Screenshot%202022-02-21%20184444.jpg" height="500">
<img src="https://github.com/LucaYan0506/Binary-code-Puzzle/blob/master/screenshot/Screenshot%202022-02-21%20184533.jpg" height="500">
 </div>
 
* Complete the puzzle  
<img src="https://github.com/LucaYan0506/Binary-code-Puzzle/blob/master/screenshot/Screenshot%202022-02-21%20195800.jpg" height="500">

* Click the "check" button.
  * If each cell of the table is green, means that solve complete the puzzle correctly
  * <img src="https://github.com/LucaYan0506/Binary-code-Puzzle/blob/master/screenshot/Screenshot%202022-02-21%20184936.jpg" height="500">

  * If some cell of the table are red, means that more than 2 identical digits are placed directly next to or below each other 
  * <img src="https://github.com/LucaYan0506/Binary-code-Puzzle/blob/master/screenshot/Screenshot%202022-02-21%20185029.jpg" height="500">
 
  * If some cell of the table are red, means that a row/column hasn't the same the number of 0s and 1s
  * <img src="https://github.com/LucaYan0506/Binary-code-Puzzle/blob/master/screenshot/Screenshot%202022-02-21%20185219.jpg" height="500">

  * If some cell of the table are yellow, means that a row/column is not unique
  * <img src="https://github.com/LucaYan0506/Binary-code-Puzzle/blob/master/screenshot/Screenshot%202022-02-21%20185029.jpg" height="500">
  
  * These information can also be seen on the application
